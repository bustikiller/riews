## Installation
Add this line to your application's Gemfile:

```ruby
gem 'riews'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install riews
```

Make sure your application is eager loading the code in the desired environments:
```ruby
  config.eager_load = true
```
